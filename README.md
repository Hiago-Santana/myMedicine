# My Medicine

O projeto visa prover ao usuário uma maneira simples e rápida de organizar seus medicamentos de acordo com as necessidades,
receitadas por um médico. Este aplicativo não visa substituir o trabalho de um profissional, mas sim auxiliar e complementar
a atuação. Organize suas medicações ao longo dos dias, semanas, meses e anos, e não se esqueça mais de administrá-los.

# Tecnologias utilizadas

- *Vite*: `v7.1.7`
- *Vue*: `3.5.22`
- *Tailwindcss*: `4.1.15`

# Como rodar

Requisitos para executar localmente

Node.js (versão recomendada: 18+)

npm ou yarn

▶️ Como rodar o projeto
# Clonar o repositório

git clone <URL-do-seu-repositório>

# Entrar no diretório

cd my-medicine

# Instalar dependências

npm install

# Rodar o app
npm run dev


Funcionalidades

Ao entrar na tela principal da aplicação, é possível ver os campos "Horários do dia", "Meus medicamentos" e "Novo medicamento".
 - Horários do dia: Neste campo, é possível ver os horários agendados para determinadas medicações predefinidas e devidamente preenchidas
 pelo próprio usuário, para o dia em questão.
 - Meus medicamentos: Aqui são listados todos os medicamentos administrados, no geral. Temos informações de dosagem, dia específico, horário, etc.
 - Novo medicamento: A área é responsável por adicionar uma nova receita ao calendário, com uma medicação que será preenchida pelo usuário.
